Question-1: What are the different ways to select an element in the DOM?

In JavaScript, there are many ways to select elements from DOM. We can use getElementById() to select a single element by its unique ID. If there are multiple elements with the same class, we can use getElementsByClassName(). To select all elements of a specific tag, like all tags, we use getElementsByTagName(). We can also use querySelector() which picks the first element that matches any CSS selector and for selecting all element we can use querySelectorAll(). These different methods let us easily work with different parts of the web page, depending on what we want to do.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: It gives the HTML code inside an element, including any tags. We can also use it to add new HTML inside an element.

innerText: It gives only the visible text from an element and respects the styling of the page.

textContent: It gives all the text inside an element, even if some parts are hidden but does not include any HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is when we add one event listener to a parent element instead of adding separate listeners to each child element.
For example, we can Imagine a list with many buttons. Instead of adding a click event to every button, we can add just one click event to the list itself and check which button was clicked.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element: const div = document.createElement("div");

To add it to the page: parent.appendChild(div);

To remove it: parent.removeChild(div);
